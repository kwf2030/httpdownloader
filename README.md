httpdownloader
==============

A simple http download tool written in Java Swing that support breakpoint transmission.

####How to build####
run
<code>
gradle jar
</code>
to generate executable JAR.

or 
<code>
gradle distZip
</code>
to generate a zip distribution.

or 
<code>
gradle installApp
</code>
to install into 'build/install'.
